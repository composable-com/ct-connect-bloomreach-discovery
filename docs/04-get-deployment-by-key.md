# Get a Deployment

```bash
curl --get https://connect.us-central1.gcp.commercetools.com/composable-product-dev-sandbox/deployments/key=orium-ct-connect-bloomreach-discovery-deployment \
--header 'Authorization: Bearer {{ token }}' | json_pp
```
